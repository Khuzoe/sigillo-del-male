# Release Checklist

1. Eseguire `npm run qa`.
2. Verificare che `assets/data/foundry.json` non sia in staging (e ignorato da git).
3. Verificare aggiornamento `assets/data/search-index.json` se sono cambiati contenuti.
4. Verificare almeno una sessione, un NPC e una missione in UI.
5. Push su branch principale.
