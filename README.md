# La Mia Wiki (Template)

Questo è un semplice template di wiki statica in HTML/CSS pensato per essere caricato su **GitHub Pages**.

## Come usarlo

1. Crea un nuovo repository su GitHub.
2. Carica questi file (index.html, pagina1.html, pagina2.html, style.css e la cartella `img/`).
3. Vai su **Settings → Pages** e abilita GitHub Pages dalla branch `main` (cartella `/root`).
4. Apri il link che GitHub ti fornisce per vedere la wiki online.

Puoi modificare liberamente i file HTML e il CSS per adattarli al tuo progetto.
